Everyday, a lot of students are preparing TOEFL. And a lot of experiences are sharing all over the internet. Is it possible to gather people together? Is it possible to let English speaking people who has free time and like to help to join? Practice is the only right way to learn a language. Not only one to one teaching. It should be real reactions between different people.

TOEFL is a very good start. Because it tests all of speaking, writing, reading and listening. There are so many articles describing how to write or how to speak. But no one is better than practice and learn from each other. So I want to build a place like StackOverFlow that people can discuss on the same question and vote to good answers. And you also can put your own article and let others to review.

FAQ

Why build a new website for TOEFL?
There are so many websites already doing this and do it very well. But I still feel confused when I start to prepare it. In China, we are trying to remember vocabularies as many as possible. We only want to pass the examination which higher score and don't care about how to use English in real life. Let learner meet the real speaker, and join different senario to learn not only vocabularies but how to live in English. So I want to build a new site to connect people around the world.
Only TOEFL and why?
TOEFL is a start. And it is good to focus on one purpose first. I think this pattern can be used to other kind of learning.
